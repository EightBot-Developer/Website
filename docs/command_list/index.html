command_list

コマンドの使い方​

コマンド表
EightBotはスラッシュコマンドと通常のコマンドの両方に対応しています。(2023年までにスラッシュコマンドに移行します。)
このチュートリアルでは、スラッシュコマンドを使って説明します。

5000​

5000兆円ほしい！を生成します。(CyberRexさんの5000choyen-apiを使用しています。)

使い方​

/5000 top: 上の文字 bottom: 下の文字

afk​

席を離れる時に使えます

使い方​

/afk reason: 理由

akinator​

アキネーターがプレイできます。



使い方​

/akinator

ban_member​

banされたメンバーを確認できます。

使い方​

/ban_member

blackjack​

ブラックジャックが出来ます。(今は英語ですが今後日本語に対応します。)

使い方​

/blackjack

bot_info​

EightBotの情報が見れます。

使い方​

/bot_info

bot_invite​

いろいろなBotを招待できます。

使い方​

/bot_invite /bot_invite bot:招待したいBotのid or メンション

downcheck​

様々なサービスの障害状況が確認できます。

使い方​

/downcheck name: サービス名

embed​

embedを作成します。



使い方​

/embed

mojiaidaspace​

一文字ごとに空白(スペース)を入れます。

使い方​

/mojiaidaspace text: テキスト

role_all_add​

全員にロールを付与します。


使い方​

/role_all_add role: ロール

role_all_remove​

全員からロールをはく奪します。


使い方​

/role_all_remove role: ロール

start​

アクティビティ(YouTubeなど)をスタートできます。

使い方​

/start activity: 選択肢からアクティビティーを選ぶ

totusi​

突然の死を生成します。(taqmさんのtotsuzen-textを使用しています。)

使い方​

/totusi text: テキスト

timeout​

ユーザーをタイムアウトします(分単位)

使い方​

/timeout user: ユーザー minutes: タイムアウトしている時間 reason: タイムアウトする理由(無くてもよい)

un_timeout​

ユーザーのタイムアウトを解除します。

使い方​

/un_timeout user: ユーザー

urlcheck​

urlが安全かチェックします。

使い方​

/urlcheck url: チェックするurl

verify​

認証パネルを作成します。








使い方​

/verify type: 足し算認証・引き算認証・掛け算認証・割り算認証から選ぶ role: ロール

urlst​

urlを短縮できます。(is.gdを使用しています。)

使い方​

/urlst url: 短縮するURL

voicetext​

googleの声が出力できます。

使い方​

/voicetext text: テキスト

youtube​

discord内でyoutubeで検索できます。

使い方​

/youtube word: 検索ワードorチャンネル名
